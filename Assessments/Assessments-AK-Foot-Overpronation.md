# 1. AK 관점의 핵심 이론
- Foot overpronation은 **후경골근(Tibialis Posterior)**의 약화와 밀접한 관련이 있음.

# 2. 진단 및 평가(Assesment)
- **근육검사** : [후경골근(Tibialis Posterior)](/Academies/applied-kinesiology/Muscle/tibialis-posterior.md)
- **충격흡수검사(shock absorber test)**
  - 검사법 : 여러 가지 벡터를 이용해 발을 두드린 뒤 지표근육의 근력을 재측정한다.
  - 지표근육 : 요근(Psoas)
  - 결과해석 : 발에 기능이상이 있으면 강했던 지표근육의 약화가 대부분 나타난다.
- **양성지지반응 유발검사**
  - 검사법 : 환자가 서있을 때 지표근육의 약화. 환자의 다리를 외회전시킨 채 서게 하여 체중이 외측장궁에 부하되게 한다. 다시 지표근육을 검사한다.
  - 결과해석 : 지표근육이 강해지면 내측장궁에 문제가 있음을 의미한다.
- **촉진**
  - 검사법 : 검사자가 내측장궁 밑으로 손가락을 넣어 긴장을 느낀다. 내측장궁을 유지시키는 인대를 촉진한다.
  - 결과해석 : 인대에 명백한 압통이 있는 경우 내측장궁의 붕괴를 의미한다.
