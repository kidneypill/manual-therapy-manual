# 진단 및 평가 (Assessment)

## Physical Exam

### 장비근 단축 검사
- **검사법** : 족배굴곡 ROM을 검사한다
- **주의사항** : 거골(Talus)에 대해 종골(Calcaneus), 주상골(Navicular), 입방골(Cuboid)을 고정할 것. 종골, 중족골을 잡아서 고정
- **정상** : 족관절에서 10도 이상의 배굴
- **비정상** : 족관절에서 10도 미만의 배굴
- **결과해석** : 비정상인 경우 **장비근 단축**을 의미한다

### Helbing's Sign
- **주의사항** : 신발을 벗은 상태에서 검사
- **양성** : 아킬레스건이 내측으로 휘어 들어간 상태

## Observation
- **기립위 관찰**
  - **주의사항** : 신발을 벗은 상태에서 검사
  - 내측족궁 높이, 거골 돌출, 전족부 외전 여부 확인
